# Farmer AI Advisory System

A comprehensive Full-Stack AI project structure for the Farmer Advisory System.

## Directory Structure

- `frontend/`: Contains the frontend UI codebase (e.g., React/Vue/Angular).
- `backend/`: Contains the backend API services (e.g., FastAPI/Flask/Django).
- `ml-model/`: Contains machine learning models, training scripts, and notebooks.
- `data/`: Raw and processed datasets used for model training and evaluations.
- `embeddings/`: Generated vector embeddings for Retrieval-Augmented Generation (RAG) pipelines or similar functionalities.

## Getting Started

### Backend
1. Navigate to the `backend` directory.
2. Create a virtual environment and install dependencies:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   pip install -r requirements.txt
   ```

### Frontend
1. Navigate to the `frontend` directory.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```
